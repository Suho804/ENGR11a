Phoenix Hand

This is the final assembly of the prostatic hand. It was printed by using Pursa mini printer for small parts and ideaMaker for the palm and the wrist.

Written assembly manual

The written manual provided a layout of the parts to assemble and it also provided compare & contrast picture of different types of screws. However, there was no identification of which pin was used for which joint nor which knuckle was for which finger. and because it was in a single color, it was difficult to locate which changes had been made.
There was no information on the length of the string which was confusing for me to build, therefore, I changed to looking at the video.

E-NABLE prosthesis

This is the e-NABLE prosthesis at 150% scale learning Open Source Hardware platform and design logic of how the hand works.
Development: For the Pins that should be attached to each joint of the hands, it was difficult to layout each part as they have subtle differences. I would change it to have a number or alphabet symbol for each pin so the user wouldn't get confused.

Video of the assembly process

For the video, I wish there were a reference photo of each incremental part for the builder to compare the parts so that they don't get confused.
I was able to make the tension string go through the hole very easily and locate the thread under the tip of the fingers. However, there was a subtle indication about the length of the string, and using a longer string to connect both two fingers to the tensioner pin was very confusing.
Reference Video on the side widget

Supplied parts Kit from 3D Universe


For the screws, the 3duniverse supplies kit had two different sizes of screws for connecting the tensioner pins, however, both of them didn't fit. I had to use an alternative screw with wider thickness: M5
The rubber band for the tension, 3duniverse supplies kit had two different strengths of tension. If it was written on the package how much tension it can provide, it would have been better.
Reference link: https://shop3duniverse.com/collections/project-kits/products/phoenix-hand-by-e-nable-assembly-materials-kit#

Functioning hands
I have modified the finger by adding a cylinder hole attached to the index finger. This functionality is a simple but important aspect to consider as one of the main use of our hands are to write. The cylinder is a tight fit to the pen's thickness, therefore, enabling the user to write efficiently.
